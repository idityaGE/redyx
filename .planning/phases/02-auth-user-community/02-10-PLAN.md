<task type="auto">
  <name>Task 1: Start full stack and verify API endpoints via curl</name>
  <files></files>
  <action>
    1. Bring up the full Docker Compose stack:
       ```
       docker compose up -d --build
       ```
    2. Wait for all services to be healthy (check with `docker compose ps`)
    3. Run API verification sequence via curl through Envoy (port 8080):

       **Auth flow:**
       ```bash
       # Register
       curl -s -X POST localhost:8080/api/v1/auth/register \
         -H 'Content-Type: application/json' \
         -d '{"email":"test@test.com","username":"testuser","password":"password123"}' | jq .

       # Get OTP from auth-service logs
       docker compose logs auth-service --tail=20 | grep "OTP"

       # Verify OTP (use code from logs)
       curl -s -X POST localhost:8080/api/v1/auth/verify-otp \
         -H 'Content-Type: application/json' \
         -d '{"email":"test@test.com","code":"<OTP_CODE>"}' | jq .

       # Login
       curl -s -X POST localhost:8080/api/v1/auth/login \
         -H 'Content-Type: application/json' \
         -d '{"email":"test@test.com","password":"password123"}' | jq .
       # Save the access_token from response
       ```

       **User profile:**
       ```bash
       # Get profile (public, no auth needed)
       curl -s localhost:8080/api/v1/users/testuser | jq .

       # Update profile (auth required)
       curl -s -X PATCH localhost:8080/api/v1/users/me \
         -H 'Content-Type: application/json' \
         -H "Authorization: Bearer <TOKEN>" \
         -d '{"displayName":"Test User","bio":"Hello, world!"}' | jq .
       ```

       **Community:**
       ```bash
       # Create community
       curl -s -X POST localhost:8080/api/v1/communities \
         -H 'Content-Type: application/json' \
         -H "Authorization: Bearer <TOKEN>" \
         -d '{"name":"testcommunity","description":"A test community","visibility":1}' | jq .

       # Get community
       curl -s localhost:8080/api/v1/communities/testcommunity | jq .

       # List communities
       curl -s localhost:8080/api/v1/communities | jq .
       ```

       **Rate limiting:**
       ```bash
       # Rapid-fire requests to trigger rate limit
       for i in $(seq 1 15); do
         curl -s -o /dev/null -w "%{http_code}\n" localhost:8080/api/v1/communities
       done
       # Should see 429 after ~10 requests (anonymous tier)
       ```

    4. If any endpoint fails, diagnose via:
       - `docker compose logs {service-name}` for server errors
       - Check Envoy admin at `localhost:9901/clusters` for cluster health
       - Verify database connectivity: `docker compose exec postgres psql -U redyx -d auth -c '\dt'`

    5. Fix any issues found and re-test.
